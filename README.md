**Combo Extractor Pro** is a lightning-fast, GUI-based bulk combo extraction and filtering tool. Built with Python and advanced multi-core processing, it is the **absolute best choice for working with a massive number of files or heavy dumps**. It allows you to scan huge datasets, extract specific lines using multiple keywords, and automatically organize them into neat, categorized files without any lag.

## 🔥 Key Features

* **📚 Heavy Dump Support:** Highly optimized to handle thousands of files and massive datasets seamlessly. The best tool for large-scale extractions.
* **🚀 Ultra-Fast Processing:** Uses advanced CPU Multi-threading (Fast Mode) for maximum extraction speed.
* **📂 Smart Auto-Sorting:** Automatically categorizes extracted data into:
  * `Mail:Pass`
  * `User:Pass`
  * `Number:Pass`
  * `Mix/Combined`
* **✂️ Password Length Filter:** Automatically filters out invalid passwords (keeps 4 to 50 characters).
* **⚙️ Custom Output Formats:** Choose between Clean Combo (`ID:PASS`), Full Combo (`URL:ID:PASS`), or Raw Dump.
* **🛡️ Built-in Update System:** Remote version checking and admin kill-switch enabled.
* **💻 Modern GUI:** Dark-themed, user-friendly interface built with CustomTkinter.

## 🛠️ How to Use

1. Download the latest `.exe` release.
2. Open the `keywords.txt` file (created automatically) and add your target keywords (e.g., `netflix.com`, `spotify`).
3. Run the program, select your input folder containing the huge dump files, and click **START EXTRACTION**.
4. Your sorted combos will be saved in a new folder named `Bulk_Results_[Date_Time]`.
